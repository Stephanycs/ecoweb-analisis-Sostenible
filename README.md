# Analisis de Sostenibilidad Web de Empresas Climaticas 

## Descripcion
Analisis exploratorio de 509 empresas climaticas a nivel mundial,
midiendo la huella digital de sus sitios web mediante emisiones 
de CO2 por visita, peso de pagina y uso de hosting verde.

## Hallazgos principales
1. El hosting verde reduce las emisiones un 26% por visita
   (1.68g vs 2.26g CO2), pero ninguna empresa llega a cero emisiones.

2. Las empresas de alimentacion (Farming, Food Waste, Food) 
   son las mas contaminantes digitalmente con mas de 3.4g por visita.

3. Los Scripts JavaScript son el componente mas pesado (29.4%),
   superando a las imagenes (25.7%) — algo contraintuitivo.

## Herramientas usadas
- Python (Google Colab)
- pandas
- matplotlib
- Excel (.xlsx)

## Estructura del analisis
1. Carga y exploracion del dataset
2. Limpieza de datos (valores vacios, tipos, categorias)
3. Analisis 1 — Hosting verde vs emisiones
4. Analisis 2 — Categorias mas contaminantes
5. Analisis 3 — Componentes del sitio web
6. Informe final con recomendaciones
